# CV - Francesc Vila Subias

Personal CV website built with [Astro](https://astro.build) and [TailwindCSS](https://tailwindcss.com).

## Features

- Print-friendly A4 layout (21cm x 29.7cm)
- Responsive design
- Two-page CV format
- PDF export ready (use browser's print to PDF)

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Tech Stack

- Astro 5
- TailwindCSS 4
- TypeScript
